What is the difference between Margin, Border, and Padding?

To put it plainly, Margin is the outermost property, Border is the middle property, and Padding is the innermost property.

How do they function?

They all have a similar function which is to provide spacing, but they are different in the sense of location and usage.

The best way of explaining this is by showing you the "box model"

The box model is a layout concept that is used to define the layout and dimensions of html elements on a webpage.

The box model can be broken into 4 seperate areas.

Margin

An invisible box that seperates html elements

▾

Border

A box that surrounds an html elements padding

▾

Padding

A box that surrounds an html elements inner content, similar effect to that of a highlighter on paper

▾

Content

The inner content of an html element, things such as <.h2> & <.p>
